DarkMode

Switch any website to dark mode. Simply paste the code provided into the JavaScript console of any website to switch it to dark mode. Test it on this site to see how it works.
Reloading the site you use this on will turn off dark mode.

How to access JavaScript console:
On Mac: option + command + c, then at the top click on "Console"
On other computers: right click and "Inspect", then at the top click on "Console"
